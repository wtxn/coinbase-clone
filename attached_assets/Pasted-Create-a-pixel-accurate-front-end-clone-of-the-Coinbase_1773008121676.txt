Create a pixel-accurate front-end clone of the Coinbase website (https://www.coinbase.com).

The goal is to reproduce the exact layout, sections, spacing, and visual style of the Coinbase homepage, but only as a frontend UI clone with no real crypto functionality.

Use React + Vite + Tailwind CSS.

The website must be fully responsive and visually match the Coinbase homepage design as closely as possible.
